CTP

컨텐츠 우측 하단에 고정되어 표시되며, 페이지 전반의 주요 행동을 제어하는 메인 버튼 역할을 한다.
수정, 목록, 삭제, 등록과 같은 핵심 기능을 제공하며, 사용자가 페이지 내 모든 주요 작업을 신속하게 수행할 수 있도록 지원한다. 여러 버튼이 함께 배치되는 경우 버튼 간 간격은 16px을 유지한다.
