Form Lable과 함께 사용하여 직관적으로 구분한다. 인풋 영역은 1개일 경우 max-width: 1000px을 사용한다.
인풋은 최대 3개까지 배치할 수 있으며 인풋간의 간격은 24px로 고정한다.

Type & Size

기본좌/우 Padding을 12px, 텍스트와 아이콘은 14px의 간격을 유지한다. 만일 인풋 영역이 작아질 경우 모든 텍스트는 말줄임처리 한다.

에러가 생성되었을시 문구생성 성공일시는 p.error태그 삭제나 hidden 처리

드롭다운Max width: 1000px / 최대 3개까지 사용

Input file

첨부 기능을 통해 업로드된 파일을 화면에 표시하는 영역이다. 사용자가 첨부한 파일 이름과 삭제 버튼(아이콘)을 함께 제공해, 필요할 때 해당 파일을 손쉽게 제거할 수 있다. 한 화면에 최대 3개까지 배치하며, 파일 간 간격은 16px로 유지한다.
