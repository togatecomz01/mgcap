반환지연금
계약이 종료 또는 해지되어 고객이 물건의 반환의무가 발생하였음에도 불구하고 반환을 지체한 경우, 고객은 반환기일 다음날로부터 반환일까지 산정한 경과일수에 대해 아래 산식에 의해 산출된 반환지연금을 지급하기로 합니다.
  • 반환지연금 = [일 리스료 + (일 리스료 X 반환지연금율)] X 경과일수
  • 일 리스료 = 월 리스료 X 12 /365
  • 예시) 일 리스료 2만원, 반환지연금율 120%, 경과일수 10일인 경우 [일 리스료 20,000원 + (일 리스료 20,000원 X 반환지연금율 120%)] X 경과일수 10일 = 440,000원
